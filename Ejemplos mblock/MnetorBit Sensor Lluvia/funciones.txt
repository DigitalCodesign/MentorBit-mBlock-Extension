Extensión para usar el modulo de lluvia de MentorBit en mblock. Cuenta con cuatro bloques, el primero es para configurar el pin al que se conecta el modulo, el segundo es para consultar si esta lloviendo o no (booleano), el tercero es para consultar el valor del sensor (valor analógico) y el ultimo es para mostrar directamente un mensaje de si se ha detectado lluvia o no (texto).
